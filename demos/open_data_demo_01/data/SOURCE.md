# Source: strain_stress_open.csv

- **Purpose:** Minimal open dataset for offline demo verification (MTR-1 pipeline).
- **License:** CC0 1.0 Public Domain (see LICENSE).
- **Content:** Strain (dimensionless) and stress (Pa) pairs in the linear elastic range. Values are consistent with typical published materials testing data in the public domain.
- **Format:** CSV with headers `strain`, `stress`. Compatible with existing MTR-1 calibration (dataset_relpath mode).
- **No network required:** This file is included in the repo so the demo runs fully offline.
